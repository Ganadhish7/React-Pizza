# React Pizza App
React pizza app is a Single Page Application(SPA).

Technology Stack:
Developed a React Pizza application using React Router for routing, Tailwind CSS for styling, and Redux for both remote and UI state management.

API Integration:
Integrated an external API to dynamically load the pizza menu, managing asynchronous operations with promises and async/await.

User Cart and Order Management:
Implemented a user-friendly interface for adding items to the cart, collecting order details, and allowing users to prioritize orders.

Order Submission and Tracking:
Enabled users to submit orders via a POST request to the API, generating unique order IDs for tracking and implementing a user-friendly lookup feature.

Challenges and Solutions:
Overcame challenges related to asynchronous operations, state management, user input validation, and responsive design through strategic use of technology and problem-solving skills.
